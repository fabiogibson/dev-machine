# dev-machine
My ubuntu machine for python development.

Installation
--------------------

type::

    sh -c "$(curl -fsSL https://raw.githubusercontent.com/fabiogibson/dev-machine/master/installer.sh)"

What will be installed
---------------------------

   * build-essential
   * python-dev
   * python-setuptools
   * python-software-properties
   * pyenv
   * openssl
   * libssl-dev
   * libpq-dev
   
**Languages**
